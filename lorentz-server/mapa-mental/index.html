    Visão geral hierárquica¶

    Lorentz Server (CTI)
│
├── 🔌 Conexões
│   ├── Central telefônica (3CX / Asterisk / Intelbras)
│   │   ├── Múltiplas centrais simultâneas
│   │   ├── Reconexão automática
│   │   └── Monitoramento de ramais e troncos
│   │
│   ├── Agentes desktop (TCP)
│   │   ├── Login/Logout
│   │   ├── Operações telefônicas
│   │   └── Chat interno
│   │
│   ├── Agentes web (banco de dados)
│   │   ├── Sem instalação de software
│   │   ├── Comunicação via tabela de situação
│   │   └── Mesmas funcionalidades do desktop
│   │
│   ├── Supervisor web (TCP)
│   │   ├── Consulta de estado dos ramais
│   │   └── Monitoramento em tempo real
│   │
│   ├── URA / IVR (TCP + XML)
│   │   ├── Screen pop
│   │   ├── Pesquisa de chamada
│   │   └── Dados de navegação
│   │
│   └── Monitor CTI (UDP)
│       ├── Consulta e alteração de parâmetros
│       ├── Visualização de estados
│       └── Comandos administrativos
│
├── 📞 Distribuição de chamadas (ACD)
│   ├── Fila de espera
│   │   ├── Ordenação FIFO (primeiro a chegar)
│   │   ├── Status em tempo real (quantidade + tempo)
│   │   └── Limite configurável de chamadas
│   │
│   ├── Fluxo de roteamento
│   │   ├── Destinos
│   │   │   ├── Grupo de agentes
│   │   │   ├── Agente específico
│   │   │   ├── Ramal fixo
│   │   │   ├── Grupo de ramais (rodízio)
│   │   │   └── Desligar chamada
│   │   │
│   │   ├── Condições
│   │   │   ├── Data e hora (=, >, <, >=, <=, <>)
│   │   │   ├── Intervalo de período
│   │   │   ├── Dia da semana
│   │   │   ├── DDD de origem
│   │   │   └── Número de telefone
│   │   │
│   │   └── Transbordo
│   │       └── Tempo máximo de espera por etapa
│   │
│   └── Seleção de agente
│       ├── Receptivo prioritário primeiro
│       ├── Maior tempo livre entre iguais
│       └── Verificação de pausa desligada
│
├── 👤 Gestão de agentes
│   ├── Estados
│   │   ├── Login
│   │   ├── Disponível (pausa desligada)
│   │   ├── Pausa eventual
│   │   ├── Pausa produtiva (com motivo)
│   │   └── Logout
│   │
│   ├── Permissões
│   │   ├── Chamadas ativas (externas)
│   │   ├── Receptivo / Receptivo prioritário
│   │   ├── Transferência
│   │   └── Chat entre agentes
│   │
│   └── Contadores em tempo real
│       ├── Chamadas (internas/externas, ativas/receptivas)
│       ├── Pausas (quantidade por tipo)
│       └── Tempos acumulados por estado
│
├── 📱 Operações telefônicas
│   ├── Fazer chamada (interna/externa)
│   ├── Atender chamada
│   ├── Desligar
│   ├── Reter chamada (hold)
│   ├── Recuperar chamada retida
│   ├── Transferir para ramal
│   ├── Consulta (sem desligar)
│   ├── Conferência (3 vias)
│   ├── Captura de chamada
│   ├── Enviar DTMF
│   └── Power Dialing (discagem preditiva)
│
├── 📊 Monitoramento
│   ├── Painel do supervisor
│   │   ├── Estado de todos os agentes
│   │   ├── Fila de espera por serviço
│   │   ├── Alertas de agentes
│   │   └── Discagem e distribuição manual
│   │
│   └── Monitor CTI externo
│       ├── Estado de agentes e ramais
│       ├── Parâmetros do servidor
│       └── Comandos administrativos
│
├── 🎙️ Gravação
│   ├── Gravação automática (por grupo/agente)
│   ├── Gravação manual (com motivo)
│   ├── Gravação de tronco
│   ├── Suporte a múltiplas placas de gravação
│   └── Motivos de gravação configuráveis
│
├── ⚙️ Configuração (arquivo .ini)
│   ├── Banco de dados
│   │   ├── Servidor, banco, usuário, senha
│   │   └── Driver ODBC
│   │
│   ├── Portas de comunicação
│   │   ├── Integração (agentes)
│   │   ├── Serviço (supervisor)
│   │   ├── URA
│   │   └── Host de escuta
│   │
│   ├── Central telefônica
│   │   ├── IP(s) da central
│   │   ├── Porta da central
│   │   └── Fabricante/Modelo
│   │
│   ├── Funcionalidades
│   │   ├── Monitorar ramais
│   │   ├── Atender tronco
│   │   ├── Mensagem de fila
│   │   ├── DISA
│   │   ├── CallBack Dialing
│   │   └── Quantidade máxima de fila
│   │
│   ├── Asterisk
│   │   ├── Habilitado
│   │   ├── Usuário AMI
│   │   └── Senha AMI
│   │
│   └── Logs
│       ├── Eventos (CSTA)
│       ├── Pacotes (agentes)
│       └── Erros
│
└── 📋 Logs
    ├── Log de erros (por data)
    ├── Log de agentes (por data)
    ├── Log de eventos CSTA
    ├── Log de URA (por data)
    └── Log de eventos gerais (por data)
